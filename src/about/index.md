# About CyberJournal

CyberJournal is a free, open-source cybersecurity resource built so young people can learn about something they're passionate about.

## Who Built This

CyberJournal was created by a first-year Computer Science student in the Netherlands who wanted a single place to learn cybersecurity concepts properly — and figured others might want the same.
